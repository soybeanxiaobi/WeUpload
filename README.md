# WeUpload

for of 解决单个上传问题

## 第一步 - 安装依赖
node环境: v10.x
1. cd client && yarn
2. cd node && yarn

执行 yarn 或者 npm install

## 第二步 - 启动 client
根目录下执行
yarn dev 或者 npm run dev


## 第三步 - 启动 node
根目录下执行
npm run node

## 第四步 - 访问页面
client: localhost:3000

可参考博客: https://mp.weixin.qq.com/s/GOAInFVzjaudPa00l2D6_w
